Метод Саати

Метод анализа иерархий, придуманный Томасом Л. Саати, — структурированный метод принятия решений, основанный на попарных сравнениях. Сравниваются и критерии, по которым проводится оценка, на предмет значимости между друг другом, и объекты по данным критерям между собой. Данный метод используется для определения приоритетов в сложных задачах с неочевидным выбором. Можно сказать, что этот метод - универсальный алгоритм принятия решений.
